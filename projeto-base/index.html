Resumo do Projeto

Esse projeto envolve a criação de um Blog Pessoal utilizando html com css com fins didáticos. Nele você aprende passo a passo, a entender o uso das tags de html e estilos de css externamente

Descrever os Recursos

Esse projeto terá 3 páginas, sendo a Inicial, Sobre e Contato. A inicial será a homepage com dados mais importantes, seguida do Sobre que terá informações do desenvolvedor e Contato na qual terá um formulário para envio de dúvidas.
